# Data Dictionary

## Field Descriptions

| Field Name | Data Type | Description |
|------------|-----------|-------------|
| Order_ID | String | Unique order identifier (ORD####) |
| Date | Date | Transaction date (YYYY-MM-DD) |
| Year | Integer | Year of transaction |
| Month | String | Month name |
| Month_Num | Integer | Month number (1-12) |
| Quarter | String | Quarter (Q1-Q4) |
| Day_of_Week | String | Day name (Monday-Sunday) |
| Category | String | Product category |
| Product | String | Product name |
| Unit_Price | Decimal | Price per unit ($) |
| Quantity | Integer | Number of units sold |
| Total_Sales | Decimal | Total transaction value ($) |
| Customer_ID | String | Unique customer identifier (CUST####) |
| Region | String | Geographic region |
| Age_Group | String | Customer age bracket |
| Gender | String | Customer gender |
| Payment_Method | String | Payment type used |
| Shipping_Method | String | Shipping option selected |
